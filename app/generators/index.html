怎样添加你的 generator »

如果你的 generator 符合下列标准，它将会自动展示到这个列表中。这个列表每隔一小时更新一次。

  • 更新到 npmjs.org 仓库
  • 有以下模块的 package.json 文件:
    • 它的名字应该定为 generator-your-unique-name。这意味着它应该以 `generator-` 开头，并且遵循 npm 的命名统一标准。作用域名称支持未来的扩展。
    • yeoman-generator 符合关键词特性。
    • 一个好的描述。如果它只是一个 generator 的分支，你应该描述它与别的分支的不同和为什么要把它作为一个分支。
  • 如果它也有以下信息这将是很好：
    • 作者信息，包括名称和链接到您的主页。
    • Generator 的文献链接网站。
    • Github 链接到你的仓库。 注意: 如果你的 Github 仓库将不能访问。您的 generator 将不会出现在列表中，直到您删除或更新这个仓库。

如果你不遵循这些准则，你的 generator 可能会被添加到黑名单。如果你的 generator 已经在这了，请解决我们通知你的问题，然后提交一个PR，以从黑名单的条目中删除。
